isv. 0.7.1

Predicates for JavaScript


is provides a slew of functions that allow you to compare values and test types. This leads to more functional, concise, legible, and correct code.

This is extremely useful when filtering data or comparing against a set of data. 

  ['venus', 'neptune', 1, []].filter(is.str); // ['venus', 'neptune']
  [1.2, 3, 5].every(is.int) // false

Make room on your utility belt next to underscore.js.

  _.filter([1, 2, 3], _.partial(is.less, 1)); // [2, 3]
  _.partition([1, -2, -3, 4], is.pos); // [[1, 4], [-2, -3]]

The project is hosted on GitHub. You can report bugs and discuss features on the issues page or send tweets to @trevor_landau.

A complemented version of the is predicate functions. All predicate functions are available in this namespace

Example

is.not.null(null) // false

instanceis.instance(Cls, [inst])

Returns a predicate that evaluates instanceof against a given object. Returns true if inst instanceof Cls is true.

ternaryis.ternary(pred, [a [b]])

If a pred is given:

Returns a function(a, b) that evaluates pred(a, b)

If pred and a and is given:

Returns a function(b) that evaluates pred(a, b)

If pred and a and b is given:

Returns the evaluation of pred(a, b)

If pred is of type boolean, a, and b are given

Returns bool ? a : b

Example

var pred1 = is.ternary(is.less);
pred1(1, 2); // 1

var pred2 = is.ternary(is.less, 1);
pred2(2); // 1

is.ternary(is.less, 1, 2); // 1
is.ternary(1 < 2, 'a', 'b'); // 'a'

Chaining

everyis.every()

alias is.all()

Returns a chainable interface for lazily executing predicates.

Invoke .val() which returns true if all predicates evaluate to true.

Example

var chain = is.every();
chain = chain.less(1, 2).fn(is.exists);
chain.val(); // true
chain.NaN(1).val(); // false

is.every().exists('mars').str('asteroid').val(); //true

someis.some()

alias is.any()

Returns a chainable interface for lazily executing predicates.

Invoke .val() which returns true if at least 1 predicate evaluate to true.

Example

var chain = is.some();
chain = chain.equal(1, 2).fn(is.exists);
chain.val(); // true
chain.NaN(1).val(); // true

is.some().exists(null).str('asteroid').val(); // true
is.some().exists(null).str(1).val(); // false

Utils

complementis.complement(fn)

aliasis.invert(fn)

Returns a function a that receives the same args as fn and returns the opposite truth value.

Used internally

Example

var fn = is.complement(is.fn);
fn(is.fn); // false

partialis.partial(fn, ...)

Returns a partially applied function based on fn.

Used internally

Example

var fn = is.partial(is.less, 1);
fn(2); // true

modis.mod(a, b)

Returns the value of a % b

Used internally

Example

is.mod(6, 5); // 1
is.mod(6, 6); // 0
